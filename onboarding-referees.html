Referee Onboarding — Rosterly

Welcome, Referee! You run game day: you open the game, set or confirm the lineups, score the match live inning by inning, and submit the final result. This guide gets you from first login to a finished game.

📷 _Screenshot placeholder: Rosterly home screen for a referee_

1. Getting in — register, then log in

Rosterly is invite-only, and everyone registers first. You'll get a referee code from your league admin.

First time — register:

  1. Open the Rosterly app and tap Create an account.
  2. Enter your phone number, your referee code, and your name.
  3. Tap Create Account — you're registered as a Referee.

Every time after — log in:

  • On the sign-in screen, enter your phone number + your referee code.
📷 _Screenshot placeholder: Register screen, then Sign-in screen_
Note: You can't log in until you've registered. The code must match your role — a code for a different role won't log you in. If a code says it's invalid, check with your admin that it's active.

2. Join your league

To score league games you need access to the league, granted by your admin.

  1. Open the Referee tab (or the menu ☰).
  2. You'll see the available leagues.
  3. Tap Request Access on your league.
  4. Your admin approves it — you'll get a notification when you're in.
📷 _Screenshot placeholder: league list with "Request Access" button_
No league needed for a fun game: you can referee a casual (non-league) game any time without league access — see step 5.

3. Game day — open the game

A game becomes playable shortly before its start time (the join window, set per league).

  1. Open the Referee tab.
  2. Find today's game. Before the join window opens it isn't joinable yet; once it opens you can pull it into a live session.
  3. Tap the game to start.
📷 _Screenshot placeholder: referee game list with a joinable game_

4. Set the lineups

Each team needs a batting order and fielding assignments. There are two sources:

  • Captain's lineup — if the captain already submitted one, it's ready to use.
  • Manual lineup — if not, you can type the lineup in yourself (player names, batting order, positions).
📷 _Screenshot placeholder: manual lineup entry_

5. Score the game live

Run the match inning by inning:

  • Track outs, balls/strikes, and the score.
  • Record batting and fielding as the game plays out.
  • The live game advances through innings until it's complete.
📷 _Screenshot placeholder: live scoring screen_

Fun (non-league) games

You can also run a casual game outside any league — no schedule, no standings, just live scoring. Start one from the Play Game / fun-game entry. Guests can referee fun games too.

📷 _Screenshot placeholder: fun game scoring_

6. Submit the final result

When the game ends:

  1. Enter the final score.
  2. Tap Submit Result.

This finishes the game, updates the league standings, and rolls up each player's stats. Captains and players are notified the game ended.

📷 _Screenshot placeholder: submit-result screen_

7. Standings

Open Standings (Home card or menu ☰) to see results and standings update as games finish across the league.

📷 _Screenshot placeholder: standings table_

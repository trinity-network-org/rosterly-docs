Guest Onboarding — Rosterly

You don't need an account to start using Rosterly. As a guest you can explore the app, referee a casual game, and follow league standings. When you're ready to join a team or league, an invite code unlocks the full experience.

📷 _Screenshot placeholder: Rosterly home screen for a guest_

1. Getting started — no code needed

  1. Open the Rosterly app.
  2. Continue as a guest (you don't have to enter a code yet).
  3. You'll land on the Home screen with an overview of what Rosterly does.
📷 _Screenshot placeholder: guest home / overview_

2. Referee a fun game

Guests can run a casual (non-league) game — no schedule, no standings, just live scoring. Great for a pickup game or trying the app out.

  1. Open Play Game (Home card or menu ☰).
  2. Start a fun game.
  3. Score it live — innings, outs, batting and fielding.
📷 _Screenshot placeholder: fun game live scoring_

3. Follow the standings

You can browse league Standings to see scores and results as games finish.

  1. Open Standings (Home card or menu ☰).
📷 _Screenshot placeholder: standings table_

4. Get full access — enter an invite code

To join a team or league — as a player, captain, or referee — you need an invite code from a captain or league admin, and you register with it (first-timers can't just log in).

  1. Open Settings (menu ☰) → Sign In → Create an account.
  2. Enter your phone number, the invite code you were given, and your name.
  3. Tap Create Account — your role unlocks. After that, you log in with your phone + that same code.
